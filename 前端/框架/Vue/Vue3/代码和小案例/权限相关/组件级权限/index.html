组件级权限

组件级这一块我们能做的事其实不多，我们要做的方向其实就是尽量地减少它的侵入性。就是不要让权限代码污染到我们写的组件。

能不能做到呢？那还是要看情况，如果说权限和组件的关联使用的是一种比较统一的方式，比方说有这个权限就能看得见，没有这个权限就看不见，它只会影响组件的可见度、或者是一些别的统一状态的话，那么这个侵入性还能减少。

如果说不是这样的，比方说它不是看得见或看不见的区别，而是点击过后要处理不同的逻辑，那就麻烦些了。我们就很难减少它的侵入性。

所以，这一块我们一般只能做一些简单的处理。

比方说以下的例子中，根据不同的角色展示不同的按钮组件。以下例子中，权限只是影响了组件的可见度，所以我们可以做一个自定义的指令或者做一个组件来实现。

组件的方式实现

我们可以写一个权限组件，然后引入该权限组件，用该组件包裹所有的按钮。

页面 index.vue：

<template>
  <div class="com-permission">
    <label for="">切换用户：</label>
    <el-select
      v-model="user"
      placeholder="请选择"
      size="large"
      value-key="value"
      style="width: 240px"
    >
      <el-option
        v-for="item in options"
        :key="item.value"
        :label="item.label" :value="item"
      />
    </el-select>

    <div class="com-permission-btns">
      <!-- 自定义的权限组件 -->
      <authority-comp
        v-for="item in btns"
        :key="item.key"
        :auth="user.permissions.includes(item.key)"
      >
        <el-button class="com-permission-btns-btn" :type="item.type">{{ item.label }}</el-button>
      </authority-comp>
    </div>
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'
import AuthorityComp from './components/Authority.vue';

defineOptions({
  name: 'ComPermission'
})

type User = {
  label: string
  value: string
  permissions: string[]
}

const user = ref<User>({
  label: '',
  value: '',
  permissions: []
})

const options = [
  {
    label: '管理员',
    value: 'admin',
    permissions: ['add', 'delete', 'edit', 'search'] // 用户拥有的权限
  },
  {
    label: '张三',
    value: 'zhangsan',
    permissions: ['add', 'edit', 'search']
  },
  {
    label: '李四',
    value: 'lisi',
    permissions: ['add', 'search']
  },
  {
    label: '王五',
    value: 'wangwu',
    permissions: ['search']
  },
]
const btns = [
  { label: '新增用户', key: 'add', type: 'primary' },
  { label: '删除用户', key: 'delete', type: 'danger' },
  { label: '修改用户', key: 'edit', type: 'warning' },
  { label: '查询用户', key: 'search', type: 'success' },
]
</script>

<style lang="scss" scoped>
.com-permission-btns {
  display: flex;
  align-items: center;

  &-btn {
    margin-right: 8px;
  }
}
</style>

权限组件 Authority.vue：

<template>
  <div v-if="props.auth" class="authority-comp">
    <slot></slot>
  </div>
</template>

<script setup lang="ts">
import { defineProps } from 'vue'

defineOptions({
  name: 'AuthorityComp'
})

type Props = {
  auth: boolean
}
const props = withDefaults(defineProps<Props>(), {
  auth: true
})
</script>
