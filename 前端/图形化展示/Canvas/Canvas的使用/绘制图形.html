绘制图形

绘制线条

在 Canvas 中绘制线条可以用到以下 API：

  • 绘制：moveTo、lineTo、stroke。
    • moveTo：表示把 画笔 移动到某个位置。
    • lineTo：表示用 画笔 画到某个位置。相当于把我们设置的 点 通过 线条 连接起来。
    • 例如我们有起始点和终点，通过 moveTo 和 lineTo 那么就可以画一条线了。
    • stroke：moveTo 和 lineTo 只是绘制命令或者说是描绘怎么样的绘制信息，真正要绘制的时候需要调用 stroke。
  • 设置样式：lineWidth、strokeStyle。
    • lineWidth：设置线条的宽度。
    • strokeStyle：设置画笔颜色。
  • 路径：beginPath、closePath。
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')
	
    // 把画笔移动到 (0, 10) 的位置
    ctx.moveTo(0, 10)
      
    // 把画笔从 (0, 10) 的位置画到 (200, 10) 的位置
    ctx.lineTo(200, 10)
      
    // 光有 moveTo 和 lineTo 是不能把页面绘制出来的，
    // 因为 moveTo 和 lineTo 只是描述怎么去绘制的信息，
    // 只有调用 stroke 才会进行绘制。
    ctx.stroke()
  </script>
</body>

</html>

<!DOCTYPE html>

我们发现页面显示的效果好像和我们想象中的不太一样，我们在代码中是想要绘制了两条线段，分别是红色的 线条1 和蓝色的 线条2。但是，为什么在页面上显示的两条线段都是蓝色的呢？

这是因为我们调用 stroke 的时候，它会把写在 stroke 上面的所有绘制指令都执行一遍。

比方说：

  1. 首先，在调用第一个 stroke 的时候，会绘制一条红色的线段。
  2. 接下来，因为 lineWidth 和 strokeStyle 在 全局中是唯一的状态，所以后面的 lineWidth 和 strokeStyle 会把之前设置的 lineWidth 和 strokeStyle 给覆盖掉。
  3. 然后在调用第二个 stroke 的时候，会把上面的所有绘制指令都执行一遍，包括第一个 stroke 前面的绘制指令，所以绘制了两条蓝色的线段。
  4. 所以，这里在调用第一个 stroke 的时候绘制了一条红色的线条，调用第二个 stroke 的时候，绘制了两条蓝色的线条。

注意，这里其实红色的线段仍然是存在的，只是被蓝色的盖住了，我们可以修改一下代码中红色线段的线宽：

  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    // 线条1
    ctx.moveTo(0, 10)
    ctx.lineTo(200, 10)
    ctx.lineWidth = 50
    ctx.strokeStyle = 'red'
    ctx.stroke()

    // 线条2
    ctx.moveTo(0, 50)
    ctx.lineTo(200, 50)
    ctx.lineWidth = 10 
    ctx.strokeStyle = 'blue'
    ctx.stroke()
  </script>
</body>

</html>

那么，怎么解决这个问题呢？我就想画一条红色的线条和一条蓝色的线条。

这里我们可以用一个 API 叫做 beginPath，它的表示新建一条路径。当 stroke 遇到 beginPath 的时候，stroke 就会知道当前的绘制指令是作用于该路径的。

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    // 线条1
    ctx.beginPath() // 新建一条路径
    ctx.moveTo(0, 10)
    ctx.lineTo(200, 10)
    ctx.lineWidth = 10
    ctx.strokeStyle = 'red'
    ctx.stroke() // 作用于当前的路径

    // 线条2
    ctx.beginPath() // 新建一条路径
    ctx.moveTo(0, 50)
    ctx.lineTo(200, 50)
    ctx.lineWidth = 10 
    ctx.strokeStyle = 'blue'
    ctx.stroke() // 作用于当前的路径
  </script>
</body>

</html>

绘制三角形

三角形其实就是绘制三条线段然后连接在一起。

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    ctx.beginPath() // 新建一条路径
    ctx.moveTo(0, 20) // 把画笔移动到 (0, 20) 的位置
    ctx.lineTo(200, 20) // 然后从 (0, 20) 画到 (200, 20) 形成直线
    ctx.lineTo(50, 200) // 再从 (200, 20) 画到 (50, 200) 形成折线
    ctx.lineTo(0, 20) // 最后回到 (0, 20) 形成闭合，最终形成三角形
    ctx.lineWidth = 1 
    ctx.strokeStyle = 'green'
    ctx.stroke()
  </script>
</body>

</html>

效果如下所示：

其实不管是三角形、矩形还是其他图形，只要是一个闭合的路径，最后一次 lineTo 都是要回到原来的位置。这样会比较繁琐，这时候我们可以借助一个 API 叫做 closePath，表示形成一个闭合的路径。

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    ctx.beginPath() // 新建一条路径
    ctx.moveTo(0, 20) // 把画笔移动到 (0, 20) 的位置
    ctx.lineTo(200, 20) // 然后从 (0, 20) 画到 (200, 20) 形成直线
    ctx.lineTo(50, 200) // 再从 (200, 20) 画到 (50, 200) 形成折线
      
    // ctx.lineTo(0, 20) // 最后回到 (0, 20) 形成三角形
    ctx.closePath() // 闭合路径
      
    ctx.lineWidth = 1 
    ctx.strokeStyle = 'green'
    ctx.stroke()
  </script>
</body>

</html>

页面效果和上述的完全一样。

填充三角形内容区域

我们在上一小节绘制一个三角形，如果想要对这个三角形的内容区域进行颜色的填充可以使用 fillStyle 设置填充颜色，并通过 fill 进行填充：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    ctx.beginPath()
    ctx.moveTo(0, 20)
    ctx.lineTo(200, 20)
    ctx.lineTo(50, 200)
    ctx.closePath()
    ctx.lineWidth = 1 
    ctx.strokeStyle = 'green'
    ctx.fillStyle = 'green' // 设置填充颜色
    ctx.fill() // 进行填充
    ctx.stroke()
  </script>
</body>

</html>

绘制矩形

根据之前绘制三角形的流程，我们可以知道想要绘制矩形，同样可以使用 lineTo 不断绘制连接，最后闭合路径，形成一个矩形。

但是，这里介绍一种新的方式，通过使用 rect 来绘制矩形。

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    ctx.beginPath()
    ctx.rect(50, 50, 200, 200) // 设置矩形四个角的位置
    ctx.stroke() // 连线各个点（角），绘制矩形
  </script>
</body>

</html>

rect 接收四个参数，前两个参数分别表示矩形左上角的 (x, y) 的坐标位置，表示从哪个位置开始画。后两个参数表示矩形的宽高。

页面效果如下图所示：

还有一种绘制矩形的方式：strokeRect。上述的代码中，绘制矩形是需要 rect 和 stroke 结合使用，而 strokeRect 可以直接帮助我们把矩形绘制出来，strokeRect 接收的参数和 rect 一样：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    ctx.beginPath()
    ctx.strokeRect(50, 50, 200, 200) // 绘制矩形
  </script>
</body>

</html>

填充矩形内容区域

如果是通过 lineTo 和 stroke 来绘制矩形的，一般是通过 fill 来进填充。

如果是通过 rect 或 strokeRect 来绘制矩形的，还可以通过 fillRect(x, y, width, height) 来进行填充：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    ctx.beginPath()
    ctx.strokeStyle = 'yellow' // 设置画笔颜色
    ctx.fillStyle = 'yellow' // 设置填充颜色
    ctx.fillRect(50, 50, 200, 200) // 填充区域
    ctx.strokeRect(50, 50, 200, 200) // 绘制矩形
  </script>
</body>

</html>

绘制弧线（圆弧、圆）

我们可以用以下两个 API 来绘制弧线，分别是：

  • arcTo(x1, y1, x2, y2, radius)。
  • arc(x, y, radius, startAngle, endAngle, anticlockwise)。

首先来看 arcTo(x1, y1, x2, y2, radius)，表示根据控制点和半径绘制圆弧路径，使用当前的 描点 (前一个 moveTo 或 lineTo 等函数的 止点)。根据当前 描点 与给定的 控制点 1 连接的直线，和 控制点 1 与 控制点 2 连接的直线，作为使用指定半径的圆的切线，画出两条切线之间的弧线路径

  • x1：第一个控制点的 x 轴坐标。
  • y1：第一个控制点的 y 轴坐标。
  • x2：第二个控制点的 x 轴坐标。
  • y2：第二个控制点的 y 轴坐标。
  • radius：圆弧的半径。

例如：

下图中蓝色的点是上一次画笔的落点（止点），另外两个红色的点是我们给定的控制点，假设下方的红色点为控制点1，左上方的红色点为控制点2。

那么就可以分别把：

  1. 蓝色的落点和控制点1进行连线。

  2. 控制点1和控制点2连线。

  3. 以这两条连接线作为 切线 画一条弧线路径。

可以通过代码进行测试：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    ctx.beginPath()
    ctx.moveTo(200, 10) // 给定义个画笔的落点
    ctx.arcTo(200, 300, 0, 10, 80) // 通过 arcTo 绘制圆弧
    ctx.lineWidth = 5
    ctx.stroke()
  </script>
</body>

</html>

我们应该都有注意到，这条弧线有一段好像是直线，如下图黄色框标注所示：

其实，这两条直线的切点之间的弧线才是我们要绘制出来的弧线。那为什么会有一小段是直线呢？这是因为它画完这条弧线之后，还需要和起点做一个连接。

接下来我们再看一下 arc(x, y, radius, startAngle, endAngle, anticlockwise)，这也是用来绘制圆弧路径的方法。

它表示圆弧路径的圆心在 (x, y) 位置，半径为 radius，根据anticlockwise （默认为顺时针）指定的方向从 startAngle 开始绘制，到 endAngle 结束。

绘制贝塞尔曲线

在我们生活中或在项目中，很多线条都不是简单的直线或者弧线，贝塞尔曲线可以用来绘制一些复杂的有规律的图形。

在 Canvas 中可以通过使用 quadraticCurveTo(cpx, cpy, x, y) 来绘制 二阶贝塞尔曲线。

它需要 2 个点。第一个点是 控制点，第二个点是 终点。起始点 是当前路径最新的点——在创建二次贝赛尔曲线之前，可以使用 moveTo() 方法进行改变。

  • cpx：控制点的 x 轴坐标。
  • cpy：控制点的 y 轴坐标。
  • x：终点的 x 轴坐标。
  • y：终点的 y 轴坐标。

⼆次⻉塞尔曲线调试⼯具。

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    ctx.beginPath()
    ctx.moveTo(100, 100) // 起始点
    
    // (200, 500) 是控制点，(400, 400) 是终点
    ctx.quadraticCurveTo(200, 500, 400, 400);
    ctx.stroke()
  </script>
</body>

</html>

二阶贝塞尔曲线 就是通过中间的控制点，控制显示不同的 二阶贝塞尔曲线。

在 Canvas 中还可以通过使用 bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y) 来绘制 三阶贝塞尔曲线。

相比于 二阶贝塞尔曲线，三阶贝塞尔曲线 多了一个控制点。

三次⻉塞尔调试⼯具。

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    // 三阶贝塞尔曲线
    ctx.beginPath()
    ctx.moveTo(20, 20) // 起点
    ctx.bezierCurveTo(20, 500, 200, 100, 500, 400);
    ctx.stroke()

    // 绘制辅点
    ctx.fillStyle = 'red'
    ctx.fillRect(20, 20, 10, 10) // 起点
    ctx.fillRect(20, 500, 10, 10) // 控制点1
    ctx.fillRect(200, 100, 10, 10) // 控制点2
    ctx.fillRect(500, 400, 10, 10) // 终点
  </script>
</body>
