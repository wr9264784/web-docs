变形

变形有三种变形方式：

  • translate：平移。

例如上图中，坐标系的原点默认是在 (0, 0) 的位置，如果想要改变坐标系的位置，就需要进行平移的操作。

在图中，坐标系横向平移了 x 的单位距离，纵向移动了 y 的单位距离。红色的坐标系就是平移后的位置。

  • rotate：旋转。

旋转就是对整个坐标系进行旋转某个角度。

  • scale：缩放。

平移

首先，我们来绘制一个 200 * 200 的红色矩形。

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')
    
    ctx.beginPath()
    ctx.fillStyle = 'red'
    ctx.fillRect(0, 0, 200, 200)
  </script>
</body>

</html>

<!DOCTYPE html>

状态保存和恢复（栈结构）

如果我们在平移完坐标系并且绘制完成后，还想从 (0, 0) 开始的位置再画一个矩形，怎么办呢？我们可以继续进行平移：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')
    
    // 平移
    ctx.translate(100, 100)
    ctx.fillStyle = 'red'
    ctx.fillRect(0, 0, 200, 200)

    // 再平移回去
    ctx.translate(-100, -100)
    ctx.fillStyle = 'blue'
    ctx.fillRect(0, 0, 200, 200)
  </script>
</body>

</html>

效果如下图所示：

这个 tanslate 其实是一个 累计操作，如果我们想在其它地方再去进行绘制的话，我们还需要在这个 (-100, -100) 的基础上继续进行平移，所以会非常麻烦。

那么有没有什么办法可以清晰地去指定平移后的坐标系，并绘制我们对应位置的图形呢？

这里，我们可以用到 状态保存，也就是说在平移之前可以通过 save() 把当前的信息做一个保存，然后平移完之后，再通过 restore() 恢复之前保存的信息，再进行后续的绘制：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')
  
    // 保存信息，它会保存上面绘制的所有信息
    // 包括图形的样式以及坐标系的位置。当前的坐标系还是以 (0, 0) 为原点。
    ctx.save()

    // 然后再平移
    ctx.fillStyle = 'red'
    ctx.translate(100, 100)
      
    // 此时绘制的图形，坐标系原点是 (100, 100)
    ctx.fillRect(0, 0, 200, 200)

    // 平移之后再恢复之前保存的信息
    ctx.restore()

    ctx.fillStyle = 'blue'
      
    // 此时绘制的图形，坐标系原点是 (0, 0)
    ctx.fillRect(0, 0, 200, 200)
  </script>
</body>

</html>

效果和之前一样：

矩阵变换

图形变换的本质就是对坐标系的原点进行再计算。通过 平移、旋转、缩放 可以帮助我们计算新的坐标系原点在哪个位置。

除了通过以上三种方式进行变形，Canvas 还提供了 矩阵变换 的形式来进行变形。

对于二维图形可以通过 3 * 3 的矩阵进行实现，三维图形可以通过 4 *4 的矩阵进行实现：

当我们想对图形进行变换的时候，就可以对变换矩阵的每一个参数进行修改，然后将图形中的所有原点和矩阵相乘就可以了。

语法：transform(a, b, c, d, e, f)，这六个参数分别对应着一种变换方式。

我们先来看下通过矩阵进行平移：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    // 默认原点的矩形
    ctx.fillStyle = 'red'
    ctx.fillRect(0, 0, 200, 200)

    // 矩阵变换默认参数
    // ctx.transform(1, 0, 0, 1, 0, 0)

    // 通过矩阵进行平移，根据文档我们可以发现，矩阵平移需要修改 e 和 f，
    // 也就是最后两个参数的值，其他参数不变。这里平移到 (200, 200) 的位置。
    ctx.transform(1, 0, 0, 1, 200, 200)
    ctx.fillStyle = 'blue'
    ctx.fillRect(0, 0, 200, 200)
  </script>
</body>

</html>

接下来再看下通过矩阵进行缩放：

合成

合成是一个比较重要的知识点，Cavans 提供了 26 种图片混排模式。

通过 globalCompositeOperation 属性，设置要在绘制新形状时应用的合成操作的类型。

用到哪种就去文档查就行了，具体查看 文档。

这里提供了几种模式的示例：

通常情况下，我们绘制了两个图形，后绘制的图形一般都会覆盖前面的图形之上。

例如，先绘制了一个红色的矩形，后绘制了一个蓝色的矩形，蓝色的矩形会覆盖红色的矩形，这里蓝色矩形进行了平移，如下图所示：

上图中的覆盖是正常的，但是有时候我们想要做一些特殊的处理，比方说我们想把后绘制的图形放到下面，这个时候就可以用到图像合成方法：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    // 默认原点的矩形
    ctx.fillStyle = 'red'
    ctx.fillRect(0, 0, 200, 200)

    // 设置合成模式
    ctx.globalCompositeOperation = 'destination-over'

    ctx.transform(1, 0, 0, 1, 100, 100)
    ctx.fillStyle = 'blue'
    ctx.fillRect(0, 0, 200, 200)
  </script>
</body>

</html>

裁剪

clip()：一般会跟路径相关的 API 进行使用，用于将当前或给定路径形成的区域进行裁剪。后面绘制的所有内容都只会显示在这个裁剪区域内。

例如我们先来绘制一个文本：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    // 绘制文本
    ctx.fillStyle = 'green'
    ctx.font = '44px Times New Roman'
    ctx.fillText('Hello Canvas', 50, 50)
  </script>
</body>

</html>

效果如下：

然后再进行裁剪：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    // clip 需要配合路径使用
    ctx.rect(0, 0, 200, 100) // 在 (0, 0) 的位置绘制一个 200 * 200 的矩形
    ctx.clip() // 裁剪

    // 后续绘制的内容都在这个裁剪区域内
    // 绘制文本
    ctx.fillStyle = 'green'
    ctx.font = '44px Times New Roman'
    ctx.fillText('Hello Canvas', 50, 50)
  </script>
</body>

</html>

效果如下：

可以看到文本内容显示的区域被进行了裁剪。
