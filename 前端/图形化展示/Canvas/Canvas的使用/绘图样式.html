绘图样式

线条样式

  • lineWidth：设置线条的宽度。

  • lineCap：设置线条两端的形状。

  • setLineDash：设置线条为虚线。

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    // 绘制线条
    ctx.beginPath()
    ctx.moveTo(50, 50)
    ctx.lineWidth = 15 // 线宽
    ctx.lineCap = 'round' // 线条两端的样式
    ctx.lineTo(300, 50)
    ctx.stroke()

    // 绘制虚线
    ctx.beginPath()
    ctx.moveTo(50, 100)
    ctx.lineTo(300, 100)
    ctx.lineWidth = 2
      
    // setLineDash 接收一个数组表示
    // 表示虚线的线段和空隙的长度在 20 40 60 不断重复
    // 一般情况下，数组只要两个数就足够了，分别表示线段和空隙的长度
    ctx.setLineDash([20, 40, 60])
    ctx.stroke()
  </script>
</body>

</html>

渐变

  • ctx.createLinearGradient(x0, y0, x1, y1)：表示 线性渐变，会根据两个给定的坐标值所构成的线段创建一个线性渐变。该方法返回一个线性 CanvasGradient对象。想要应用这个渐变，需要把这个返回值赋值给 fillStyle 或者 strokeStyle。
  • ctx.createRadialGradient(x0, y0, r0, x1, y1, r1)：表示 径向渐变，根据参数确定两个圆的坐标，绘制 放射性渐变的方法。这个方法返回 CanvasGradient。

通过代码实现 线性渐变：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    ctx.beginPath()

    // 设置线性渐变以及渐变方向
    const gradient = ctx.createLinearGradient(0, 0, 200, 0)
    // 设置渐变颜色
    // 处了 0 和 1 还可以在 0.x 的位置设置颜色（0.x 表示 0 到 1 区间的数）
    gradient.addColorStop(0, 'green') // 起始位置颜色
    gradient.addColorStop(1, 'blue') // 结束位置颜色
    ctx.fillStyle = gradient // 把渐变设置给填充样式
    ctx.fillRect(50, 200, 200, 100)
  </script>
</body>

</html>

设置 径向渐变：

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    // 绘制虚线
    ctx.beginPath()

    // 设置径向渐变
    const gradient = ctx.createRadialGradient(150, 300, 100, 150, 300, 50)
    // 设置渐变颜色
    gradient.addColorStop(0, 'green') // 起始位置颜色
    gradient.addColorStop(1, 'blue') // 结束位置颜色
    ctx.fillStyle = gradient // 把渐变设置给填充样式
    ctx.fillRect(50, 200, 200, 200)
  </script>
</body>

</html>

纹理样式

createPattern(image, repetition)：把图片作为样式填充纹理。

  • image：作为重复图像源的 CanvasImageSource 对象。可以是以下值：
    • HTMLImageElement (``)
    • HTMLVideoElement (``)
    • HTMLCanvasElement (``)
    • CanvasRenderingContext2D
    • ImageBitmap
    • ImageData
    • Blob
  • repetition：指定如何重复图像。有以下值：
    • "repeat" (both directions)
    • "repeat-x" (horizontal only)
    • "repeat-y" (vertical only)
    • "no-repeat" (neither)
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Canvas</title>
</head>

<body>
  <canvas id="Canvas" width="800" height="600" style="background-color: #c1c1c1;"></canvas>

  <script>
    const canvas = document.getElementById('Canvas')
    const ctx = canvas.getContext('2d')

    // 绘制纹理
    const img = new Image()
    img.src =  '../assets/img.png'
    img.onload = () => {
      const pattern = ctx.createPattern(img, 'repeat')
      ctx.fillStyle = pattern
      ctx.fillRect(50, 50, 300, 300)
    }
  </script>
</body>

</html>
