SVG 的描边动画

两个重要属性

我们先编写一段简单的 SVG 元素：

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>SVG描边动画</title>
  <link rel="stylesheet" href="./index.css">
</head>
<body>
  <svg class="icon" width="200" height="200">
    <line class="p" x1="0" y1="50%" x2="100%" y2="50%"></line>
  </svg>
  <script src="./index.js"></script>
</body>
</html>

对应的样式为：

.p {
  stroke: #f40;
  stroke-width: 10;
}

在页面中的效果为：

那么怎么在 SVG 中如何来进行描边呢？我们会需要借助两个属性：

  • stroke-dasharray：这是一个数字，表示一条线条每一段的间隔。
  • stroke-dashoffset：这是一个数字，表示实线和虚线的偏移量。如果值为负数往右边移动，值为正数往左边移动。

我们先来设置 stroke-dasharray 看下效果：

.p {
  stroke: #f40;
  stroke-width: 10;
  stroke-dasharray: 10;
}

效果如下：

可以看到通过设置了 stroke-dasharray，之前的线条变成了虚线。这个虚线每一小段之间的间隔就是我们设置的 10。 stroke-dasharray 设置的值越大，那么虚线线段之间的间隔就会越大。

我们再来设置 stroke-dashoffset 看下效果：

.p {
  stroke: #f40;
  stroke-width: 10;
  stroke-dasharray: 10;
  stroke-dashoffset: 0;
}

我们在先样式中设置为 0，然后在浏览器控制台中调整对应的值，查看页面的效果：

结合两个属性进行动画

如果把 stroke-dasharray 设置为整条线的宽度，目前整条线的宽度为 200，因为 svg 标签中的 width 设置为了 200：

.p {
  stroke: #f40;
  stroke-width: 10;
  stroke-dasharray: 200;
  stroke-dashoffset: 0;
}

效果如下：

可以看到，stroke-dasharray 设置为了整条线的宽度后，又变成了一条实线了。这是为什么呢？因为这条线条其实是一个虚线，只不过现在虚线中实的部分和虚的部分都为 200，而整个 SVG 的图标宽度为 200，所以目前只看到了实的部分，虚的部分超出范围，看不见。

然后把整条线往左移动，移动多少呢？移动一个 SVG 图标的实际宽度：

.p {
  stroke: #f40;
  stroke-width: 10;
  stroke-dasharray: 200;
  stroke-dashoffset: -200;
}

这样一来，线条实的部分就看不见了，SVG 图标变成了空白。那么我们怎么让它出现呢？我们可以写一个动画：

.p {
  stroke: #f40;
  stroke-width: 10;
  stroke-dasharray: 200;
  stroke-dashoffset: 200;
  animation: stroke 2s forwards;
}

@keyframes stroke {
  to {
    stroke-dashoffset: 0;
  }
}

现在一个简单的描边效果就出来了，效果如下：

现在这个代码我们还可以稍微做一些处理，比方说这里的 200 这个数值，很多地方都用到，这个数值对应的是 svg 标签里 line 标签的长度。

因为目前这个 line 标签的长度是从起始点 x1 到终点 x2 为 100%，刚好等于整个 svg 设置的 width 的值，所以是 200。

这样就意味着我们写的样式中的数值要始终保持和 svg 标签中设置的宽度相同，因此我们最好不要在样式中写死，而是要动态地去读取这个线的长度。

所以，我们可以使用 css 变量：

.p {
  stroke: #f40;
  stroke-width: 10;
  stroke-dasharray: var(--l);
  stroke-dashoffset: var(--l);
  animation: stroke 2s forwards;
}

然后通过去设置这个变量，就相当于同时设置了 stroke-dasharray 和 stroke-dashoffset。接下来的问题就是如何得到 SVG 图标里边线条的长度了，这时候就需要使用 JS 代码了。

// 拿到 svg 图标下所有的 line 元素，目前这里只有 1 条
const paths = document.querySelectorAll('.icon .p')

paths.forEach(path => {
  const len = path.getTotalLength() // 获取当前 path 的长度
  path.style.setProperty('--l', len)
})

页面效果不变：

.p {
