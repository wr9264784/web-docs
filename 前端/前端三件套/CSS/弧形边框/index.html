弧形边框

当我们在开发页面的时候，有时候可能会遇到弧形边框的场景，比如一些 tabbar 选项卡，或者一些侧边栏的选中样式，如下：

首先我们准备一个元素，这个元素的左上角和右上角设置为圆角，如下所示：

假设该元素标签的 class 为 tab-item， 则对应的样式如下：

.tab-item {
  display: flex;
  align-items: center;
  justify-content: center;
  width: 150px;
  height: 40px;
  font-size: 16px;
  color: #fff;
  background-color: #007BFF;
  border-radius: 10px 10px 0 0;
}

然后我们想办法在这个元素的左下角和右下角的位置分别拼接一个弧形，于是我们可以通过伪元素的方式进行添加：

.tab-item {
  position: relative;
  display: flex;
  align-items: center;
  justify-content: center;
  width: 150px;
  height: 40px;
  font-size: 16px;
  color: #fff;
  background-color: #007BFF;
  border-radius: 10px 10px 0 0;
}

.tab-item::before,
.tab-item::after {
  content: '';
  position: absolute;
  width: 10px;
  height: 10px;
  bottom: 0;
  background-color: #000;
}

.tab-item::before {
  left: -10px;
}

.tab-item::after {
  right: -10px;
}

为了方便观察，我把两个伪元素的颜色设置为了黑色，目前的效果如下：

那么我们要怎么做才能把这两个元素做成一个弧形的效果呢？我们可以使用 渐变。

比方说我们现在要对左边的元素做一个弧形的效果，可以使用 径向渐变：

.tab-item::before {
  left: -10px;
  /* 径向渐变，颜色由蓝色到红色 */
  background: radial-gradient(blue, red);
}

效果为：中心点为蓝色，然后向四周扩散为红色。如下所示：

我们还可以设置这个渐变的圆心，把这个圆心的位置设置为左上角：

.tab-item::before {
  left: -10px;
  /* circle at 0 0 表示把圆心设置为左上角 */
  background: radial-gradient(circle at 0 0, blue, red);
}

那么效果就为以左上角为中心，向四周扩散为红色，如下所示：

那么此时，我们如果改下 渐变的颜色，使得它从透明颜色到选项卡的背景颜色：

.tab-item::before {
  left: -10px;
  background: radial-gradient(circle at 0 0, transparent, #007BFF);
}

效果如下所示：

我们还可以设置 渐变的位置，透明颜色从最开始一直变到 10 个像素，然后选项卡的背景颜色是从改颜色一直变到 10 个像素：

.tab-item::before {
  left: -10px;
  background: radial-gradient(circle at 0 0, transparent 10px, #007BFF 10px);
}

效果如下：

可以看到，弧形边框的效果已经出来了。那么具体是怎么做到的呢？其实就是左下角这个元素从左上角先来 10 个像素范围的透明，然后从第 10 个像素开始，又变成这个背景颜色。所以就出现这个弧形的效果。

同理，我们可以把右边的元素也设置一下，右边的元素唯一不同的就是圆心的设置：

.tab-item::after {
  right: -10px;
  background: radial-gradient(circle at 100% 0, transparent 10px, #007BFF 10px);
}
