文字下划线动画

有以下的下划线动画效果：

这个下划线随着鼠标的移入和移出会有一个动画，这个下划线是随着文字来进行移动的，它是贴着文字来进行移动。这里我们需要怎么做呢？

我们要思考的第一个问题是，这个下划线是随着文字的方向来进行移动的，那么在样式中什么东西会随着文字来进行呢？就是文字到哪，下划线就到哪。

好像就一个东西符合，就是背景并且是 行盒的背景，块盒不行。

index.css：

.word-underline {
  margin: 15% auto;
  width: 400px;
  line-height: 2;
}

.word {
  background: red;
}

但是这个背景和下划线有什么关系呢？我们是不是可以把这个背景变成一张背景图，背景图可以设置，我们只需要把尺寸给它设置窄一点，不就变成下划线了。

所以，我们可以把这个背景变成 渐变背景。因为 渐变背景 本质上就是一个 背景图：

.word {
  /* 默认渐变方向是从上到下，这个改成了从左到右 */
  background: linear-gradient(to right, #ec6b5f, #61c554)
}

效果如下：

接下来就需要设置一下背景图的尺寸：

.word {
  background: linear-gradient(to right, #ec6b5f, #61c554) no-repeat;
  background-size: 100px 2px;
}

效果如下：

线条出来了，接下来就是改变背景图的位置：

.word {
  background: linear-gradient(to right, #ec6b5f, #61c554) no-repeat;
  background-size: 100px 2px;
  background-position: left bottom;
}

效果如下：

接下来就是要怎么实现移入和移出时的动画了，我们可以让这个背景图一开始的宽度是 0，当鼠标移入的时候宽度变成 100%：

.word {
  background: linear-gradient(to right, #ec6b5f, #61c554) no-repeat;
  background-size: 0 2px;
  background-position: left bottom;
    
  /* 只给 background-size 设置过渡效果 */
  transition: background-size 1s;
}

.word-underline:hover .word {
  background-size: 100% 2px;
}

效果如下：

但是和我们的最终动画效果还有些许差别，我们最开始演示的效果中，鼠标移入的时候，下划线是从左边开始进入，移出的时候，下划线也是从左边开始消失的。

但是上诉的样式中鼠标移入的时候，下划线是从左边开始进入，移出的时候，下划线是从右边开始消失的。

为什么会这样呢？因为这个背景图它是 background-position: left bottom; 的位置，而鼠标移出的时候 background-size 是从 100% 变成 0 的，由于这个背景图是靠左的，所以就会从右边开始消失，因为它始终保持 left bottom 的位置。

我们只需要给背景一开始设置的位置为 right bottom，鼠标移动的时候改成 left bottom 即可。

完整样式 index.css：

.word-underline {
  margin: 15% auto;
  width: 400px;
  line-height: 2;
}

.word {
  background: linear-gradient(to right, #ec6b5f, #61c554) no-repeat;
  background-size: 0 2px;
  background-position: right bottom;
  transition: background-size 1s;
}

.word-underline:hover .word {
  background-position: left bottom;
  background-size: 100% 2px;
}
