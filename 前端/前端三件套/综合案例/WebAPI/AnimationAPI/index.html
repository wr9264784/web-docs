Web Animation API

前言

在前端开发中，我们如果要做动画，一般有两种思路：

  • 第一种就是使用 css 来实现。
  • 第二种就是使用 js 来实现。

css 的实现动画比较简单，效率也很高。但是它不灵活，因为如果说它出现一些跟交互关联非常强的动画 css 就很难实现了。

对于 js 动画，官方又没有给我们提供相应的 api，我们得自己写，或者是使用一些第三方库。自己写比较麻烦，用第三方库又会导致要引入更多的 js 文件，在效率上也不是很合适。

用 js 写动画的问题在于没有一个很好的原生动画 api，但是这种情况现在已经解决了，目前官方给我们提供了一套 api，叫做 Web Animation API。该 api 应用于元素，然后运行动画。

这套 api 就可以让我们在 js 里边去实现动画，而且它的效率跟 css 动画效率差不多。

基本使用

element.animate(keyframes, options)
  • 这里的 element 指的是标签元素。
  • keyframes：关键帧对象数组，或一个关键帧对象（其属性为可迭代值的 数组）。参见 关键帧格式 以了解详细信息，其实就和我们在写 css 动画时，指定的关键帧一样。
  • options：代表动画持续时间的整数（以毫秒为单位），或者一个包含一个或多个时间属性（在 KeyframeEffect() options 参数 和下方列出）的对象。

注意：js 中的这种 api 方式实现的动画之所以效率高是因为它在执行动画的过程中不会改变元素的样式，和 css 动画一样。

这就和我们使用的第三方动画库不一样了，第三方动画库就必须要去更元素的 style 样式，这样就一定会导致主线程的重新渲染。

案例实现

以下我们通过该 api 来实现一个小案例，首先来看一下这个小案例的页面效果：

index.html：

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Web Animation API</title>
  <link rel="stylesheet" href="./index.css">
</head>
<body>
  <div class="ball"></div>

  <script src="./index.js"></script>
</body>
</html>

index.js：

// 这一块代码是在用户点击鼠标的时候，有一个光圈扩散的效果，和本节动画 api 无关
window.addEventListener('click', (e) => {
  const pointer = document.createElement('div')
  pointer.classList.add('pointer')
  pointer.style.left = `${e.clientX - 18}px`
  pointer.style.top = `${e.clientY - 18}px`
  document.body.appendChild(pointer)
  pointer.addEventListener('animationend', () => {
    pointer.remove()
  })
})

const ball = document.querySelector('.ball')

/**
 * 把小球放到屏幕中间
 */
const init = () => {
  const ballRect = ball.getBoundingClientRect()
  const w = ballRect.width / 2
  const h = ballRect.height / 2
  const x = window.innerWidth / 2 - w
  const y = window.innerHeight / 2 - h
  ball.style.transform = `translate(${x}px, ${y}px)`
}

init()

/**
 * 监听鼠标点击事件，让球运动到点击的位置
 */
window.addEventListener('click', (e) => {
  const x = e.clientX
  const y = e.clientY
  move(x, y)
})

/**
 * 小球带有动画的效果移动到指定的坐标
 * @param {*} x 
 * @param {*} y 
 */
const move = (x, y) => {
  const ballRect = ball.getBoundingClientRect()
  const initX = ballRect.left
  const initY = ballRect.top

  // 求反正切，小球缩放后的旋转弧度
  const rad = Math.atan2(y - initY, x - initX)
  // 弧度转角度
  const deg = rad * 180 / Math.PI

  // 在 animate 的 keyFrames 一个数组，每一项就是一个关键帧，
  // 我们可以不指定每一项的进度百分比，因为它默认是自动平均分配的。
  const keyFrames = [
    // 第一个关键帧，移动到初始位置
    {
      // rotate(${deg}deg) 表示旋转，根据小球当前位置和鼠标点击的位置确定旋转的角度
      transform: `translate(${initX}px, ${initY}px) rotate(${deg}deg)`,
      easing: 'ease-out', // 时间函数，设置动画的速度曲线
    },
    // 第二个关键帧，移动到初始位置，并根据鼠标位置进行缩放
    {
      // rotate(${deg}deg) 表示旋转，根据小球当前位置和鼠标点击的位置确定旋转的角度
      // 这里要注意一下变形的顺序，要先旋转，再缩放
      transform: `translate(${initX}px, ${initY}px) rotate(${deg}deg) scaleX(1.5)`,
      offset: 0.6 // offset 就相当于 css 中 keyframe 动画的 百分比，0 - 1 表示百分比

    },
    // 第三个关键帧，移动到指定位置，并根据鼠标位置进行缩放
    {
      // rotate(${deg}deg) 表示旋转，根据小球当前位置和鼠标点击的位置确定旋转的角度
      // 这里要注意一下变形的顺序，要先旋转，再缩放
      transform: `translate(${x}px, ${y}px) rotate(${deg}deg) scaleX(1.5)`,
      offset: 0.8,
      easing: 'ease-in', // 时间函数，设置动画的速度曲线
    },
    // 第四个关键帧，移动到指定位置
    {
      // rotate(${deg}deg) 表示旋转，根据小球当前位置和鼠标点击的位置确定旋转的角度
      transform: `translate(${x}px, ${y}px) rotate(${deg}deg)`
    }
  ]

  // animate 的 options 有两种配置方式，一种是直接写一个数值，表示动画的总时间
  // 另一种是写一个对象
  const options = {
    duration: 500, // 动画总时间
    fill: 'forwards' // 动画结束后，保持最后一帧的状态
  }

  // 每次移动都会开启一个新的动画，这会有性能问题，
  // 因为之前的动画并不会消失。它会把过去生成的动画以及它们的配置全部都保留。
  // 通过 console.log(ball.getAnimations()) 我们可以看到所有的动画.
  // console.log(ball.getAnimations())

  // 如果说之前的动画已经不需要了，我们可以手动清除掉
  ball.getAnimations().forEach(animation => animation.cancel())
  
  // 找到小球的元素，然后通过小球元素调用 animate()
  ball.animate(keyFrames, options)
}

index.css：

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  background-color: #000;
}

.pointer {
  position: absolute;
  left: 0;
  top: 0;
  width: 36px;
  height: 36px;
  border: 1px solid #fff;
  border-radius: 50%;
  animation: warn 0.38s ease-in;
}

@keyframes warn {
  0% {
    transform: scale(0);
    opacity: 0;
  }

  25% {
    transform: scale(0.25);
    opacity: 0.5;
  }

  50% {
    transform: scale(0.5);
    opacity: 1;
  }

  75% {
    transform: scale(0.75);
    opacity: 0.5;
  }

  100% {
    transform: scale(1);
    opacity: 0;
  }
}

.ball {
  width: 60px;
  height: 60px;
  border-radius: 50%;
  background-color: pink;
}
