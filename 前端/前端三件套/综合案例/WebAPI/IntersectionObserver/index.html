IntersectionObserver

前言

IntersectionObserver 提供了一种异步观察目标元素与其祖先元素或顶级文档视口（viewport）交叉状态的方法。我们可以通过观察目标元素与视口是否有交叉的情况来编写图片懒加载的效果。

如下所示：

可以看到，当图片元素不在视口区域时，显示的是默认的加载中的图片，而当我们拖动滚动条时，底部的图片元素进入视口，与视口产生了交叉，那么就会把默认的加载中图片替换成真实的图片，从而实现图片的懒加载效果。

代码实现

index.js：

/**
 * IntersectionObserver 接口（从属于 Intersection Observer API）
 * 提供了一种异步观察目标元素与其祖先元素或顶级文档视口（viewport）交叉状态的方法。
 * 其祖先元素或视口被称为根（root）。
 */

/**
 * 创建观察器，观察交叉情况
 * 第一个参数是回调，这个回调函数的运行时间点就是交叉情况发生变化的时候。
 * 第二个参数是配置，什么情况下才算交叉，就需要通过这个配置项进行判断了。
 */
const ob = new IntersectionObserver((entries) => {
  // 一开始观察的时候，不管有观察了有多少个元素，多少个元素发生了交叉，
  // 都会把所有的交叉情况合并，不会反复执行这个回调函数，合并之后可以
  // 通过传递的参数 entries 获取每一个观察对象的交叉情况。
  // entries 获取的是所有观察对象元素，不管是交叉的还是没有交叉的。
  // console.log(entries);

  for (const item of entries) {
    if (item.isIntersecting) { // 值为 true 表示发生交叉
      // 用真实图片的地址覆盖原先的加载中图片
      item.target.src = item.target.dataset.src

      // 优化操作，图片显示出来后就可以不需要再观察这个元素了
      ob.unobserve(item.target) // 取消观察
    }
  }
}, {
  // 表示观察器观察的元素和哪个元素发生交叉，
  // 只能配置观察元素的父元素，或者父元素的父元素，也就是祖先元素。
  // 如果配置为 null 或者不传的话，表示观察元素与视口的交叉情况。
  root: null,

  // 针对 root 这个元素，对它的观察范围进行扩散或者是收缩，
  // 值为正数表示扩散，就是对 root 元素的周边进行扩散，
  // 负数表示收缩，就是对 root 元素的周边进行收缩。
  // 所有的偏移量均可用像素（px）或百分比（%）来表达，
  // 默认值为 “0px 0px 0px 0px”。
  rootMargin: '0px',

  // 表示阈值，取值范围 0 -> 1，
  // 就是我们观察的元素和这个 root 重叠多少的时候，会触发回调执行。
  // 元素进入 root 或离开 root 都会回调的执行。
  // 如果为 0，表示观察元素和 root 刚好碰到就触发。
  // 如果为 1，表示观察元素要完整的进入 root，才会触发回调。
  threshold: 0
})

// 找到所有带有 data-src 属性的图片元素
const imgs = document.querySelectorAll('img[data-src]')

imgs.forEach(img => {
  // 开始观察元素
  ob.observe(img)
})

createImgs.js：

const listDom = document.querySelector('.img-list')

// 创建图片元素
function createImgs() {
  // 循环 50 次，每次生成 10 张图片
  for (let n = 0; n <= 10; n++) {
    for (let i = 1; i <= 10; i++) {
      // 生成图片的 src 路径
      const src = '../assets/images/' + i + '.jpeg' // 真实显示的图片
      const defaultSrc = '../assets/images/loading.gif' // 默认的图片（加载中图片）
      const img = document.createElement('img')

      img.src = defaultSrc // 设置图片默认加载中图片
      img.dataset.src = src // 设置 data-* 属性
      img.classList.add('img-item')

      // 把图片添加到容器中
      listDom.appendChild(img)
    }
  }
}
